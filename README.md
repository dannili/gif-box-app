# gif-box-app
An ember project that displays today's top trending gifs using giphy API

[![]https://media.giphy.com/media/2zlSwREmLmiQw/giphy.gif)]()

# Running the app
Clone
```shell
clone https://github.com/dannili/gif-box-app.git
```

Install dependencies
```shell
npm install
```

Start server
```shell
ember s
```

Fun time!
```shell
http://localhost:4200/
```
